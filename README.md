## Original dataset of Brazilian Soccer Database 

https://www.kaggle.com/datasets/ricardomattos05/jogos-do-campeonato-brasileiro/data

The [2012-2025.csv](./2012-2025.csv) file provides complementary data to the original dataset, specifically covering processed records from the 2022 through 2025 seasons.


## Original dataset of Brazilian Football Dataset [2003 - 2024]

https://www.kaggle.com/datasets/lucasyukioimafuko/brasileirao-serie-a-2006-2022/data?select=teams.csv

The [team-2012-2025.csv](./team-2012-2025.csv) file contains geographical information for the teams in the [2012-2025.csv](./2012-2025.csv) dataset. Team names have been sanitized to ensure a perfect match between both files.


## Original dataset of Matches Brazilian Football [2003 - 2025]

https://www.kaggle.com/datasets/rczekster/matches-brazilian-football-from-2003-to-2019/data?select=ranking-2003-2025.txt

The [ranking-2012-2025.csv](./ranking-2012-2025.csv) file contains the final ranking of each team in all season from 2012-2025. Team names have been sanitized to ensure a perfect match between both files.
